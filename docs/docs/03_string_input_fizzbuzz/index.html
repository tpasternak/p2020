    String, Input, Hurra #

    Typ String #

    Zmienne typu int

    a = 5
print(type(a))  # >> 'int'

    Zmienne typu string

    a = "5"
print(type(a)) # >> 'str'

    Konkatenacja - operatorem + możemy łączyć zmienne typu string

    a = "Dzień "
b = "dobry"
print(a + b) # >> Dzień dobry

    Konwersje - funkcjami int i str możemy konwertować typy zmiennych

    a = "1"
b = "1"
print(a + b) # wypisze 11
print(int(a) + int(b)) # wypisze 2
c = int(a) + int(b)
print("liczba: " + str(c)) # żeby połączyć liczbę int z tekstem, trzeba ją zamienić na str

    Interpolacja - możemy wstawić zawartość zmiennej bezpośrednio do stringa

    a = 5
b = f"2+3={a}"
print(b)

    Mnożenie

    a = "ab" * 5
print(a) # >> ababababab

    Można też użyć znaku ', który jest równoważny "

    a = 'ab'
print(a)

    Potrójne ''' lub """ pozwala na zdefiniowanie tekstu składającego się z wielu linii

    a = '''
To jest tekst
podzielony na wiele linii
'''
print (a)

    Hurra! #

    Zadanie https://sio2.mimuw.edu.pl/c/oij14-1/p/hur/ z XIV OIJ

    n = int(input())
i = 1
while i <= n:
    if i % 7 == 0 and i % 11 == 0:
        print("Wiwat!")
    elif i % 7 == 0:
        print("Hurra!")
    elif i % 11 == 0:
        print("Super!")
    else:
        print(i)
    i = i + 1
