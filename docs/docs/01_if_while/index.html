    Arytmetyka # 

    print(1 + 5)       # Wypisz wynik dodawania 1+5
print(5 % 2)       # Wypisz resztę z dzielenia 5/2
print(5 % 2 == 0)  # Wypisz, czy reszta z dzielenia 5/2 jest równa 0

    Operatory #

    • 2 + 2
    • 2 * 2
    • 2 ** 2
    • 10 % 3 - reszta z dzielenia (10 % 3 == 1)
    • 10 / 3 - dzielenie (10 / 3 == 3.33333)
    • 10 // 3 - dzielenie z zaokrągleniem w dół (10 // 3 == 3)

    Zmienne #

    Podstawową jednostką, na której operują programy w pythonie są zmienne. Używamy ich do przechowywania wartości.

    a = 1
print(a)

    Warto zwrócić uwagę, ze operator = nie oznacza równania, jak w matematyce, ale przypisanie.

    a = 1
b = 1
c = a + b
print(c)
a = 5
print(c)

    Co ciekawe, dzięki temu, następujące wyrażenie może mieć sens:

    a = 1
a = a + 1
print(a)

    If/Else #

    Czasami zachodzi potrzeba wykonania kodu tylko w przypadku, kiedy nasze zmienne mają pewne właściwości. Wykorzystujemy do tego konstrukcję if/else

    a = 155
if a % 2 == 0:
    print(f"{a} jest parzysta")
else:
    print(f"{a} jest nieparzysta")

    While #

    Pętli while używamy, kiedy chcemy wykonać kod wielokrotnie. Przeanalizujmy kilka przykładów jej użycia:

    • Wypisz wszystkie liczby od 0 do 100
    n = 0                 # Zaczynamy z n równym 0
while n <= 100:       # Wykonuj tak długo, jak długo n jest mniejsze od 0
    print(n)          # Wypisz n
    n = n + 1
    • Wypisz wszystkie liczby mniejsze od 100, ale podzielne przez 3
    n = 0
while n < 100:
    if n % 3 == 0:
        print(n)
    n = n + 1
    • Wypisz sumę liczb od 0 do 100
    suma = 0
n = 0
while n <= 100:
    suma = suma + n
    n = n + 1
print(suma)
    • Wypisz sumę liczb mniejszych lub równych 100 i podzielnych przez 3
    suma = 0
n = 0
while n <= 100:
    if n % 3 == 0:
        suma = suma + n
    n = n + 1
print(suma)
    • Sprawdź, czy liczba jest pierwsza
    liczba = 157
n = 2
pierwsza = True
while n < liczba:
    if liczba % n == 0:
        pierwsza = False
    n = n + 1

if pierwsza:
    print("Jest pierwsza")
else:
    print("Nie jest pierwsza")
