    Liczby pierwsze - kontynuacja # 

    z = 1
while z < n:
    z = z + 1
    if n == z:
        print("Ta liczba jest liczbą pierwszą")
        break  
    if n % z == 0:
        print(f"Ta liczba nie jest liczbą pierwszą. Jest podzielna przez {z}")
        break

    Co to jest funkcja? #

    Funkcja to podprogram, którego używamy do podzielenia naszego właściwego programu na mniejsze części.

    Funkcje składają się z

    • nazwy (suma)
    • argumentów(a, b, c)
    • zawartości return (a + b + c) * 2
    def suma(a, b, c):
    return (a + b + c) * 2

    Teraz możemy przenieść kod sprawdzający, czy liczba jest pierwsza do nowej funkcji i nazwać ją jest_pierwsza.

    def jest_pierwsza(n):
    z = 1   
    while z < n:
        z = z + 1
        if n == z:
            return True  
        if n % z == 0:
            return False

i = 1
while i <= 100:
    if jest_pierwsza(i):
        print(i)
    i = i + 1
